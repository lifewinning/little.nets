  -] DolDoc
    DolDoc is the propriatary TempleOS document type.  The editor, compiler, 
    assembler and whole operating system seamlessly handle DolDocs.  Cmds like 
    Dir() and Grep() can output links to the cmd line.
  -] Burning CD/DVDs
    * Prepare a directory with the files you wish to burn.
    
    * Use CDFileCreate() to make an ISO image file.
    
    * Use CDImageWrite() to burn an ISO file onto a CD or DVD.
    
    * See Making a Distro ISO.
-] Level 2: Admin Help
  Use Mount() to mount disk drives, perhaps, if you are booted to a LiveCD.
  
  You recompile the kernel with MakeOSPrtBootInstall(), and specify which drives 
  should be mounted at boot.  See ::/Misc/MakeDistro.CPP.
  
  Installing
  
  ::/Kernel/Adam1.CPP the TempleOS equivalent of the AUTOEXEC.BAT.  It is run at 
  start-up by the Adam Task.
